Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 randomly-selected group of cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, humidity, cloudiness and wind speed. This site provides the source data and visulizations created as part of the analysis, as well as explanationsand descriptions of any trends and correlations witnessed.
